Variáveis em JavaScript

Em JavaScript, existem três palavras-chave usadas para declarar variáveis: let, var e const.

let

A palavra-chave let é usada para declarar variáveis ​​com escopo de bloco. As variáveis criadas com let só podem ser acessadas dentro do bloco onde foram criadas. Isso significa que se você criar uma variável let dentro de um bloco de código, ela não estará disponível fora desse bloco.

      if (true) {
        let x = 5;
        console.log(x); // 5
      }
      console.log(x); // erro: x não está definido

var

A palavra-chave var é usada para declarar variáveis com escopo de função ou escopo global. As variáveis criadas com var podem ser acessadas em qualquer lugar da função onde foram criadas. Se uma variável var for criada fora de uma função, ela terá escopo global e poderá ser acessada em qualquer lugar do seu código.

      function teste() {
        var x = 5;
        console.log(x); // 5
      }
      console.log(x); // erro: x não está definido
    
      var x = 5; // variável global

      function teste() {
        console.log(x); // 5
      }
      teste();

const

A palavra-chave const é usada para declarar variáveis com valores constantes. Uma vez que uma variável é atribuída com const, seu valor não pode ser alterado. Variáveis criadas com const têm escopo de bloco, assim como as criadas com let.

      const PI = 3.14;
      PI = 3; // erro: PI não pode ser alterado
    
      if (true) {
        const x = 5;
        console.log(x); // 5
      }
      console.log(x); // erro: x não está definido
